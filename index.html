Summer 2026 · Hipral Group

Build your own tiny code judge.
Then get a computer to prove your code is correct.

Over five mornings you'll build Mini-Judge: a small tool that checks whether code is correct, and use AI assistance every step of the way. You'll vibe-code with a real AI and test what it gives you, learn Rust, then go further than testing ever can: writing a formal specification and using a real tool called Verus to prove your code is correct for every possible input, with an AI trying its hand at the spec and the proof too. By Friday, you'll run a brand new problem through both pipelines yourself. Everything runs in the browser: Python via Pyodide, Rust and Verus via their official playgrounds. No installs needed.
