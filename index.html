Build your own tiny code judge.
Then get a computer to prove your code is correct.

Over five mornings you'll build Mini-Judge: a small Rust tool that checks whether code is correct. On Day 3 you'll go further than testing ever can: you'll write a formal specification and use a real tool called Verus to prove your code is correct for every possible input, not just the ones you tried. By Friday, you'll use all of this to run a real experiment on an AI: does a precise, formal spec make it more likely the AI's code actually verifies? Everything runs in the official, browser-based Rust and Verus playgrounds. No installs needed.
